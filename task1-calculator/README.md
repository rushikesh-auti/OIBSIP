# OIBSIP - Task 1: Calculator

This is the first task of the **OIBSIP Internship**, where a simple **calculator** application is created using **HTML**, **CSS**, and **JavaScript**.

## Project Overview

The calculator provides basic arithmetic functions:
- Addition
- Subtraction
- Multiplication
- Division

The application features a responsive design, ensuring it works seamlessly on both mobile and desktop devices.

## Project Structure


task1-calculator/
│
├── index.html

## Features

- **Responsive Design**: The layout adapts to various screen sizes (mobile, tablet, desktop).
- **Basic Operations**: Includes addition, subtraction, multiplication, and division.
- **Clear Button**: Allows users to reset the display.
